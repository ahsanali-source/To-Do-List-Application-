# Simple GUI To-Do List App

A clean and interactive desktop-based To-Do List application built in Python using the Tkinter Graphical User Interface (GUI). 

## Key Features
* **Easy Adding:** Type a task and add it instantly to your list.
* **Smart Validation:** Shows a pop-up warning if you try to add an empty task.
* **Mouse Selection:** Easily select any task with your mouse to remove it when completed.
* **Visual Progress Bar:** Watch your progress bar move from 0% to 100% dynamically as you clear tasks.

## Prerequisites
This application runs entirely on Python's built-in libraries (**Tkinter**). You do **NOT** need to install any external libraries or packages. Just ensure you have Python installed on your system.

## File Structure
Place your script file in your project folder:
* `ToDoList.py` (The main Python GUI file)

## How to Run the Application

Follow these simple steps to start the application:

1. **Open your terminal** or command prompt (cmd).
2. **Navigate** to the folder where your `ToDoList.py` file is located.
3. **Run the script** by executing the following command:
   ```bash
   python ToDoList.py
